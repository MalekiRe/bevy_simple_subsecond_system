## Installation

Install the Dioxus CLI of the newest alpha build:
```sh
cargo binstall dioxus-cli@0.7.0-alpha.0
```

Make sure you're not using LD, see https://github.com/DioxusLabs/dioxus/issues/4144

## Usage

Add the plugin, use `#[hot]`, run with

```sh
dx serve --hot-patch 
```

## Examples


Run the example with
```sh
dx serve --hot-patch --package hotpatching_ui
```
